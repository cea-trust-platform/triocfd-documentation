TrioCFD Tutorials
=================


In this section, you will find tutorials for using TrioCFD.

If you have never used TrioCD of TRUST, you can start by visiting the [TRUST documentation](https://cea-trust-platform.readthedocs.io/en/latest/index.html). 
The [README](https://github.com/cea-trust-platform/trust-code/blob/master/README.md) on GitHub will guide you for installing TRUST.

For every tutorial, you are expected to start from an empty directory and to have sourced the TrioCFD environment (see :ref:`quickstart-target`). You will be reminded of this during the tutorials.
