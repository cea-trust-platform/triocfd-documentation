Detailed Description

Cette classe implemente un operateur de portance interfaciale

de la forme F_{n_l} = - F_{k} = C_{n_l, k} (u_k - u_n_l) x rot(u_n_l) ou la phase n_l est la phase liquide porteuse et k une phase gazeuse le calcul de C_{n_l, k} est realise par la hierarchie Portance_interfaciale_base
