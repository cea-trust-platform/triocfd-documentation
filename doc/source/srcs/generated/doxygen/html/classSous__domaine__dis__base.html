Detailed Description

Les classes qui heritent de Sous_domaine_dis_base doivent surcharger la methode discretiser();

L Sous_Domaine definit un sous-ensemble des elements du Domaine. De meme, le Sous_domaine_dis_base definit un sous_ensemble des attributs du Domaine_dis, en particulier des faces.
