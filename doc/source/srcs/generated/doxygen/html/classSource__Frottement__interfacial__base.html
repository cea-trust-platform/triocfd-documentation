Detailed Description

Cette classe implemente un operateur de frottement interfacial

de la forme F_{kl} = - F_{lk} = - C_{kl} (u_k - u_l) le calcul de C_{kl} est realise par la hierarchie Coefficient_Frottement_interfacial_base
