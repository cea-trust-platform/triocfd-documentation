Detailed Description

Cette classe definie l'interface generique d'une matrice telle qu'elle est utilisee dans Trio-U. En consequence elle n'est pas instanciable. Toutes les types de matrices devront deriver de cette classe de base et implementer ses methodes abstraites.

Dans les commentaires des methodes A represente un objet Matrice_base.
