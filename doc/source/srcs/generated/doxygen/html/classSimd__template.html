Detailed Description

template<typename _TYPE_>
class Simd_template< _TYPE_ >

Functionalities provided by the class are :

  • load vector size aligned data from memory (SimdGet)
  • getting x[i-1] and x[i+1] for finite difference algorithms (SimdGetAtLeft, SimdGetAtRight, etc)
  • arithmetic operations (+ - * /)
  • conditional affectation (SimdSelect) See simd_malloc() and simd_free() to allocate aligned blocs of memory.
