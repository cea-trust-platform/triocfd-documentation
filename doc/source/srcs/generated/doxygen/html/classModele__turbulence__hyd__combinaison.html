Detailed Description

et (ou) des variables d espace et de temps pour l equation de Navier Stokes La syntaxe a respecter est la suivante : Modele_turbulence combinaison { nb_var nb_sources_a_specif fonction ... turbulence_paroi ... } -Mot cle nb_var : nb_sources_a_specif est a preciser par l utilisateur : correspond aux nombre de champs sources qui interviendront dans l expression du modele (0 par defaut) -Mot cle fonction : Une expression doit etre specifiee par l utilisateur f(x,y,z,t,nom_champs_sources) -Mot cle turbulence_paroi : Choix d une loi de paroi (eventuellement negligeable) a preciser par l utilisateur
