Detailed Description

"Transport_Interfaces_FT_disc" Le champ frontiere contient une valeur d'angle de contact exprimee en degres (voir methode readOn) Attention: l'heritage de "Symetrie" est necessaire pour

  • le calcul du gradient de l'indicatrice (nul au bord)
  • le calcul du champ indicatrice_som du fichier lml pour les cas-tests de non regression
