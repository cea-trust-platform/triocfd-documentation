End the count of a counter and update values in the Stat_Internals object of the Statistiques object.

Arret du compteur counter_id.

Parameters
counter_idThe ID of the counter that the user want to start. It is a global variable that is declare in stats_counters.cpp
quantityA user defined quantity which depends on the counter (can be a number of iteration, a buffer size,.... It is set to zero by default @count Parameter that count the number of time the counter has been used between begin_count and end_count. By default equal to 1.
track_commIndicate if the user want to record the communication associated with the counter, set to true by default

On ajoute quantity a la somme des 'quantity' (par defaut 0) stockees pour ce compteur et on ajoute 'count' (par defaut 1) au compteur d'appels a end_count pour ce compteur (quantity sert a sommer par exemple la taille des donnees traitees pendant ce comptage et count sert a compter combien de fois on a appele ce compteur)
