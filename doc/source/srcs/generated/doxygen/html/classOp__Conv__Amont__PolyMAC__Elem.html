Detailed Description

Cette classe represente l'operateur de convection associe a une equation de transport d'un scalaire. La discretisation est PolyMAC Le champ convecte est scalaire Le schema de convection est du type Amont L'iterateur associe est de type Iterateur_PolyMAC_Elem L'evaluateur associe est de type Eval_Amont_PolyMAC_Elem
